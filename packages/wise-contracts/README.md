# wise-contracts

Pydantic models and schema references for canonical interface contracts defined in
`docs/architecture/canonical/03-canonical-architecture.md` §7:

- Discovery Record (JSON-LD)
- Ingest Package (BagIt + PREMIS)
- Preserved Object Descriptor (OAIS AIP)
- Entity Assertion (RDF triples)
- Quality Annotations

Contract implementations are added in Reference Capability 1 implementation phases.
