# RC16 Public Demonstration Release

> **Public demonstration only. No payment processing enabled. No checkout. No order creation. No fulfillment. No customer data collection.**

This folder packages the lowest-risk RC15 revised artifacts into a public demonstration release. It is intended for editorial preview, stakeholder review, and non-transactional public demonstration only.

## Included artifact groups

- Ebook sample: `artifacts/ebooks/big-cats-of-the-world/manuscript.md`
- Audiobook sample: `artifacts/audiobooks/big-cats-of-the-world/narration-script.md`
- YouTube scripts: `artifacts/youtube/`
- Pinterest briefs: `artifacts/pinterest/`
- Product page copy: `artifacts/products/`

## Source policy

All artifacts derive from RC15 revised content under `content/revised/`. The release intentionally excludes not-ready original drafts identified in `docs/implementation/rc15-publication-quality-report.md`.

## Rights policy

No artifact grants image, music, source, or commercial-use rights. Every visual, audio, thumbnail, product mockup, caption, and credit line must be reviewed before production publication.

## Commerce policy

This release does not enable payments, checkout, order creation, fulfillment, customer accounts, customer data collection, or product availability claims.

## Architecture/governance scope

RC16 follows ADR-011. It does not modify governance, architecture, agents, or ADRs.
