# Architecture Registry

**Purpose:** Maintain the authoritative list of approved architectures.

This document is the single source of truth for institutional architecture. All systems, programs, and initiatives must align with an architecture listed here. Draft or superseded architectures carry no institutional authority.

## Governance

New architectures are added to this registry only after completing the change control process defined for each entry. Superseded architectures remain listed for historical reference with status updated accordingly; they must not be cited as current authority.
