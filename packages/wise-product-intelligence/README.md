# wise-product-intelligence

Reference Capability 7 product intelligence package.

This package provides deterministic product recommendation data for commercial
validation surfaces. It operates inside the existing implementation/package
boundary and does not modify Architecture v1.0, governance, agents, or ADRs.

Output:

- `recommended_products.json`

Commercial intelligence fields:

- `product_category_score`
- `product_fit_score`
- `estimated_giftability`
- `estimated_repeat_purchase`
